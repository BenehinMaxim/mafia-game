# Мафия — Ведущий

Игра Мафия для ведущего. PWA-приложение для iOS и Android.

## Установка на iPhone

1. Откройте Safari по ссылке: https://benehinmaxim.github.io/mafia-game/
2. Нажмите кнопку "Поделиться" (квадрат со стрелкой вверх)
3. Прокрутите вниз и выберите "На экран «Домой»"
4. Нажмите "Добавить"

## Особенности

- Автосохранение игры
- Работает офлайн
- Поддержка iOS (add to home screen)
- Автоматическое распределение ролей
